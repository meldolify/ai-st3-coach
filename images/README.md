# Clinical Images Directory

This folder contains clinical images for ST3 interview scenarios.

## How to Add Images

1. Save your clinical images (X-rays, photos, etc.) in this folder
2. Use descriptive filenames that match the scenario, e.g., `nec_fasc.jpg`
3. Supported formats: JPG, PNG, GIF

## Image Naming Convention

Each scenario can have an associated image file. The image filename should match the scenario:

- **Necrotising Fasciitis**: `nec_fasc.jpg`
- **Digital Amputation**: `digital_amp.jpg`
- **Chemical Burn**: `chemical_burn.jpg`
- etc.

## Example Images Needed

For the Necrotising Fasciitis scenario (`nec_fasc.txt`), place a clinical photo showing necrotising fasciitis as `nec_fasc.jpg` in this folder.

The image will automatically display when that scenario is selected in the simulation room.

## Note

Images are loaded from your local drive and are NOT included in this repository for privacy/copyright reasons. You need to source appropriate clinical images yourself.
